Step-by-Step Guide

1. Update and Upgrade System Packages
Ensure that your system packages are up-to-date.

2. Install Apache, MySQL, and PHP
Install the LAMP stack (Linux, Apache, MySQL, PHP) on your Raspberry Pi.

3. Start and Enable Apache and MySQL Services
Ensure that the Apache and MySQL services are running.

4. Secure MySQL Installation
Run the security script to remove insecure default settings.
Follow the prompts to set up the root password and secure your MySQL installation.

5. Create the Database and User
Log in to MySQL and create the database and user.

6. Import the Database

7. Set Up the Project Directory
Create a directory for your project in the Apache web directory.
Ensure the directory structure is correct.

8. Set Correct Permissions
Set the appropriate permissions for the web directory.

9. Configure the Database Connection
Ensure your `includes/getMovie.php` file has the correct connection details.


10. Test the Application
Open a web browser on your Raspberry Pi or another device on the same network and navigate to:
```
http://<your-raspberry-pi-ip-address>/mymovielist
```
You should see the homepage of the movie listing application.
